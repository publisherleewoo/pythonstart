# type

## 숫자(int(정수형), float(실수형))

```python
a=3
b=4

print("type",type(a))
print(a+b)

# 몫
print("몫 3//4",a//b)

# 나머지
print("나머지 3%4",a%b)

# 나머지
print("제곱 3**4",a**b)
```

## 문자열(str)

```python
# 문자열
print("test")
print('test')
print("""test   a
abc
""")
```
